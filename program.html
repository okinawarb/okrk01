13:50 ゲスト講演 「Rubyを使って3人/日でアプリをリリースしよう」
by @masuidrive

昨年のゴールデンウィークを使って開発したMarkdown対応のオンラインメモ帳サービス"wri.pe"の開発手法について解説します。

「個人サービスだからと言われないものを」という想いでサーバー、設計、実装、UI、デザイン、マーケティングまで、下記の様な技術を使いほぼ一人で行いました。

この中に一つでも興味のあるものが、あれば聞きにきてみませんか？

  • Rails4.0 + Ruby2.0
  • Heroku
  • Capybara + RSpecを使った自動テスト
  • Backbone.js
  • HTML5 AppCache + Storage
  • Google App Script
  • Analytics Measure API
